Telecom Operator Performance Analysis

Python | Tableau | Business Analytics

Análisis de desempeño operativo para identificar operadores ineficaces, llamadas perdidas, tiempos de espera y oportunidades de mejora.

Problema de negocio

El objetivo del proyecto fue analizar el desempeño de operadores en una empresa de telefonía virtual, identificando patrones de ineficiencia en llamadas entrantes, llamadas salientes, tiempos de espera y llamadas perdidas.

Preguntas clave

  • ¿Qué operadores presentan mayores niveles de ineficiencia?
  • ¿Qué días concentran mayor volumen de llamadas?
  • ¿Qué clientes tienen más llamadas perdidas?
  • ¿Cómo se comporta la duración promedio de las llamadas?

Herramientas utilizadas

  • Python
  • Pandas
  • Tableau
  • Data Cleaning
  • KPI Analysis
  • Business Insights

Metodología

Se realizó limpieza y validación de datos, análisis exploratorio, creación de KPIs, segmentación de operadores por desempeño y construcción de un dashboard ejecutivo para facilitar la toma de decisiones.

Principales hallazgos

  • Se identificaron operadores con altos niveles de llamadas perdidas.
  • Algunos clientes concentraban mayores tiempos de espera.
  • El volumen de llamadas variaba significativamente según el día.
  • Los KPIs permitieron priorizar acciones de mejora operativa.

Recomendaciones de negocio

Se recomienda monitorear los operadores con mayor número de llamadas perdidas, revisar la asignación de turnos en días de alto tráfico y usar el dashboard como herramienta semanal de seguimiento para supervisores.
